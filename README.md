# Barakat Admin

Separate Next.js admin project for managing listings, employees, and services.

Run locally:

```powershell
cd admin
npm install
npm run dev
```

Admin app:

```text
http://localhost:3001
```

Public API:

```text
http://localhost:3001/api/listings
http://localhost:3001/api/employees
http://localhost:3001/api/services
```

Data is stored in `data/*.json`.
